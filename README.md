# Lab3
# Функціональність програми
- створення об’єктів класу `GameCharacter` з використанням різних конструкторів;
- інформування користувача про те, який саме конструктор був викликаний;
- збереження об’єктів у колекції `List<GameCharacter>`;
- перегляд, пошук та видалення об’єктів;
- демонстрація перевантаженої поведінки (method overloading).

# Реалізація перевантаження
У класі `GameCharacter` реалізовано:
- **три перевантажені конструктори**, включаючи конструктор без параметрів;
- виклик одного конструктора з іншого за допомогою `this(...)`;
- **перевантаження public-методу `Attack()`** з різними сигнатурами:
  - `Attack()`
  - `Attack(int power)`
  - `Attack(string skill)`

# Як запустити програму
1. Відкрити проєкт у Microsoft Visual Studio.
2. Запустити консольний застосунок (Start або Ctrl + F5).
3. Скористатися меню для створення об’єктів різними конструкторами
   та демонстрації перевантаженої поведінки.
